
This project builds an off-chain prototype of a DAO (Decentralized Autonomous Organization) platform.

DAO Platform (Off-chain): a platform where users can create and join decentralized organizations, vote on proposals, and manage treasury information. 
Users can create DAOs, join existing ones, submit and vote on proposals, and view decision outcomes.



Using:
- Frontend & Backend: Next.js with TypeScript
- Database: PostgreSQL 

**Tech Stack:**
- Frontend & Backend: Next.js with TypeScript
- Database: PostgreSQL
- Deployment: AWS EC2
